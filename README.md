# Search_Engine

--> Google arama motorunda arama yapılırken, aranılan kelime yanlış yazıldığında kullanıcı “Bunu mu demek istediniz? …” Şeklinde bir mesajla yönlendirilir. Klavyeden girilen yanlış bir kelimeyi words.txt dosyası içindeki kelimeler ile karşılaştırarak en benzer olan kelimeyi öneri olarak benzeme oranı ile beraber ekrana yazdıran programı yazmamız istenmektedir.
